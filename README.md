* 有**启发意义**的问题/答案
* 有**科普意义**的问题/答案
* **总结整理**型的问题/答案
* 和**论文相关**的问题/答案
* 能**开阔视野**的问题/答案
* **脑洞大开**型的问题/答案
* **其他**的有趣的问题/答案
